# StegaPic
Telegram bot for picture stenographic

При использовании в качестве телеграм бота убедитесь что вставили свой токен в *main.py*

Если же вы желаете использовать методы из *encrypt.py* и *decrypt.py* без оболочки бота, тогда
можете вызвать отдельно.

#### Аргументы метода в *encrypt.py*:

| аргумент         | значение                                  |
|------------------|-------------------------------------------|
| path_img         | путь к начальному изображению             |
| text             | текст кодируемый в изображение            |
| keys_file_path   | путь для сохранения файла с ключом        | 
| encrypt_img_path | путь для сохранения итогового изображения |

#### Аргументы метода в *dencrypt.py*:

| аргумент         | значение              |
|------------------|-----------------------|
| path_img         | путь к изображению    |
| keys_file_path   | путь к файлу с ключом |
